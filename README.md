# Event Registration Website

A simple open-source event registration system built using HTML, CSS, and JavaScript. It is designed for small clubs, student communities, and local event organizers who need a lightweight and easy-to-use registration solution.

## Features

- Simple event registration form  
- Clean and responsive UI  
- Easy to customize for different events  
- Works on any static or shared hosting environment  
- No complex setup required  

## Tech Stack

- HTML  
- CSS  
- JavaScript  

## Purpose

This project is intended to help small clubs and organizations quickly set up an online registration system for events such as seminars, workshops, science fairs, and community programs.
